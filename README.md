# Rectly

**Perspective correction made simple.**

Rectly is a browser-based perspective correction tool that lets you straighten documents, whiteboards, photos, and other images without installing software.

All image processing happens locally in your browser.
Your images never leave your device.

> **Coming soon:** GitHub Pages

---

## Why Rectly?

Many perspective correction tools require uploading images to a server.

Rectly takes a different approach.

Every operation is performed entirely within your browser, allowing you to correct images while keeping them private.

No uploads.
No accounts.
No installation.

---

## Features

* Automatic corner detection
* Manual corner adjustment
* Adjustable aspect ratios
* PNG and JPEG export
* Original filename preservation
* Light / Dark / Standard themes
* Guide line customization
* Browser-based processing

---

## Privacy

Your privacy is a core design principle.

* Images are never uploaded.
* Images are never stored.
* All processing is performed locally in your browser.

---

## Supported Browsers

* Google Chrome
* Microsoft Edge
* Safari

Mobile browsers will be officially supported after additional testing.

---

## Philosophy

Rectly is built around a simple idea:

**Powerful perspective correction should be simple, private, and available directly in your browser.**
